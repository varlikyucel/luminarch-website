About LuminArch

LuminArch: a single-player architectural design game where creativity meets science. Players design simple structures using walls, windows, and sunshades—then see how their choices affect sunlight, glare, and shadow in real-time. Simulation scores daylight performance using real-world sun data.

Built for architecture students and enthusiasts, LuminArch turns daylight design into an engaging, educational challenge.

LuminArch is a serious game by Varlık Yücel (PhD thesis), supervised by Assoc. Prof. Dr. Sema Alaçam, ITU Architectural Design Computing.
